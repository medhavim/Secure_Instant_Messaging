------------------------------------------------------------------------------
Server Commands
------------------------------------------------------------------------------
To Run:
python Server.py

To Exit:
exit
quit

------------------------------------------------------------------------------
Client Commands
------------------------------------------------------------------------------
To Run:
python Client.py

To Logout:
-----------
logout
exit (shortcut: e)
quit (shortcut: d)

To List online users:
---------------------
list (shortcut: l)

To send message to other users:
--------------------------------
send <online_user_name> <message>
(shortcut: s <online_user_name> <message>)
